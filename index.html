Foobar

This Ruby program is an adaptation of a classic technical interview problem which displays sequences of the Foobar pattern. 

puts "How many items in a pattern would you like to see? "
      digits = gets.chomp.to_i
      list=[]
      (1..digits).each do|ui|
      if ui % 3 == 0 &&  ui % 5 == 0
          list.<< "FOOBAR"
       elsif ui  % 3 == 0
          list.<< "FOO"
       elsif ui  % 5 == 0
          list.<< "BAR"
       else
          list.<< ui
       end
